# biblio-dashboard

A dashboard for managing and reviewing long text bibliographic records from a bibtex file, including detailed information such as titles, abstracts (up to 300 words), publication and author information. The dashboard should also accommodate a thumbnail image for a related PDF. The layout should prioritize readability and easy navigation.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/biblio-dashboard.git
cd biblio-dashboard
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
